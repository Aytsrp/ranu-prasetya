My Recent Work's

Unit Layanan Aduan Surakata Website

This website was my final project in vocational high school, where we were assigned to build a website using a framework. I chose Next.js and MySQL with full CRUD functionality. The site was developed for the Surakarta city government to serve as a platform for handling public complaints and service requests.

Vespa Balap Indonesia Website

This website is a personal side project, a company profile built for the Indonesian Vespa community. It's made with Next.js and includes both animation and UI libraries to give it a more dynamic. I created it not just to sharpen my skills, but also to contribute something cool and functional for a community I admire.

Rantau.id Website

This website was a project for the Systems Analysis and Design, where we were tasked with designing everything from the business process to the final product. Rantau.id is a platform that helps people find temporary housing from kosts and apartments to villas and homestays. It's designed to make the search, communication, and booking process simple and seamless for both property owners and renters.

Meteor Shower Game

This game was a project for the Object-Oriented Programming, where we were assigned to create a game using Java with Greenfoot. Meteor Shower is a retro-style arcade game where the player controls a spaceship defending Earth by shooting down incoming meteors. It's a fun way to apply OOP concepts like inheritance, polymorphism, and event handling in an engaging and visual way.

Peppermint OS

Peppermint OS was assigned as part of the Cloud Computing course, where we were tasked with installing and exploring the operating system. The goal was to get familiar with lightweight Linux environments and understand how they can support cloud-based workflows.
