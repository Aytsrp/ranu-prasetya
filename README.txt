## ğŸŒ Website Portfolio - Ranu Riang Prasetya

    Website ini dibangun dengan pendekatan desain yang unik dan berbeda dari tampilan umum (antimainstream). Proses pembuatan melibatkan penggunaan Adobe Photoshop untuk merancang aset visual seperti latar belakang (background) dan logo, sehingga tampilan lebih personal dan original. Website ini juga memiliki custom font yang saya download sendiri dan juga menggunakan favicon generator pada internet untuk mengubah logo menjadi icon pada tab website. Dalam proses pengembangan, bantuan dari AI juga digunakan sebagai referensi dan solusi saat mengalami masalah, mengingat pengetahuan saya tentang Bootstrap masih terbatas. Website ini membuat saya belajar banyak tentang kustomisasi yang bisa dilakukan Bootstrap. Website ini juga memiliki css yang saya buat sendiri seperti class glass untuk membuat efek transparan pada card, dan juga untuk kustomisasi default class dari bootstrap sesuai keinginan saya.

## â— Masalah yang Saya Hadapi

    Beberapa kendala yang dihadapi dalam proses pembangunan website antara lain adalah pemahaman terhadap cara kerja class `img-fluid` pada Bootstrap, serta penerapan animasi baik melalui CSS maupun fitur animasi bawaan Bootstrap yang masih belum sepenuhnya saya kuasai. Oleh karena itu, bagian animasi belum dapat diimplementasikan secara optimal.

    Website ini tetap diupayakan untuk memenuhi kriteria penilaian UTS mata kuliah Pemrograman Website, serta mengikuti prinsip responsive design agar dapat diakses dengan baik melalui berbagai platform.

## ğŸ§© Komponen Bootstrap

    Website ini menggunakan komponen Bootstrap berikut:

        - Navbar: Digunakan sebagai menu navigasi utama yang menghubungkan antar halaman (Home, About, Portfolio, Contact). Responsive akan muncul hamburger icon pada navbar.
        - Card: Digunakan pada halaman Portfolio untuk menampilkan proyek-proyek secara terstruktur dengan gambar, judul, dan deskripsi singkat. Responsive akan mengubah layout dari card itu sendiri.
        - Carousel: Digunakan pada menu hobbies untuk menunjukkan beberapa gambar hobi, memberikan sentuhan interaktif dan visual yang dinamis pada halaman.
        - Button: Diterapkan pada elemen seperti submit form pada halaman Contact, dan aksi lainnya, dengan gaya Bootstrap agar konsisten dan responsif.
        - Popover: Digunakan pada bagian teknologi yang saya gunakan. Saat menekan logo akan muncul nama teknologi tersebut.
