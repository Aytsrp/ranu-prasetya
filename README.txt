# Website Portofolio - Ranu Riang Prasetya

Website ini dibangun dengan pendekatan desain yang unik dan berbeda dari tampilan umum (antimainstream). Proses pembuatan melibatkan penggunaan Adobe Photoshop untuk merancang aset visual seperti latar belakang (background) dan logo, sehingga tampilan lebih personal dan orisinal. Dalam proses pengembangan, bantuan dari AI juga digunakan sebagai referensi dan solusi saat menghadapi kendala, mengingat pengetahuan saya mengenai framework Bootstrap masih terbatas.

## Masalah yang Saya Hadapi

Beberapa kendala yang dihadapi dalam proses pembangunan website antara lain adalah pemahaman terhadap cara kerja class `img-fluid` pada Bootstrap, serta penerapan animasi baik melalui CSS maupun fitur animasi bawaan Bootstrap yang masih belum sepenuhnya saya kuasai. Oleh karena itu, beberapa bagian animasi belum dapat diimplementasikan secara optimal.

Website ini tetap diupayakan untuk memenuhi kriteria fungsional dan estetika, serta mengikuti prinsip responsive design agar dapat diakses dengan baik melalui berbagai perangkat.

## 🔗 Halaman Website

- Home: Berisi perkenalan singkat, foto pribadi, dan sambutan.
- About: Menjelaskan tentang diri saya lebih dekat, minat.
- Portfolio: Menampilkan 6 proyek dari mata kuliah lain atau proyek yang pernah saya kerjakan. lengkap dengan gambar, judul, dan deskripsi.
- Contact: Formulir kontak sederhana dengan input Nama, Email, Subjek, dan Pesan.

## 🧩 Komponen Bootstrap

Website ini menggunakan minimal 4 komponen Bootstrap berikut:

- Navbar: Digunakan sebagai menu navigasi utama yang menghubungkan antar halaman (Home, About, Portfolio, Contact). Responsif dan otomatis menyesuaikan tampilan pada perangkat mobile.
- Card: Digunakan pada halaman Portfolio untuk menampilkan proyek-proyek secara terstruktur dengan gambar, judul, dan deskripsi singkat.
- Carousel: Dimanfaatkan untuk menampilkan slideshow atau galeri gambar, memberikan sentuhan interaktif dan visual yang dinamis pada halaman.
- Button: Diterapkan pada berbagai elemen seperti tombol navigasi, submit form pada halaman Contact, dan aksi lainnya, dengan gaya Bootstrap agar konsisten dan responsif.

## 📱 Responsif

Website dirancang responsif untuk tampilan mobile dan desktop menggunakan Bootstrap Grid System.
